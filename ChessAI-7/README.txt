Joshua Jones (jjo108@u.rochester.edu)

Compile: javac *.java
Run: java Pawntastic 

Follow the prompts and enter the number that correspond to the options
(e.g. enter 5 for a 5x5 board)
Note that if you are using dark mode, the white chess pieces will are 
Actually the black color (formatting issues).


This program highlights using Minimax with some heuristics in order to
Play optimally against an opponent. User can also choose to play against
A random moving agent. Minimax analyzes all possible steps, but sometimes 
Fails to take the optimal move.
